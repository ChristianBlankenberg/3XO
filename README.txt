Neuronal Net for 3XO
--------------------

1. The neuronal Net is a 3 Layer NN : 9->400->9 represented by wih.csv & who.csv
2. wih.csv & who.csv are calculated by Python project : https://github.com/ChristianBlankenberg/NN_Python/blob/main/3XO_Network_Training.ipynb
3. The python project is trained by nnTrainingData.txt which is created by iteration overall possible boars 
and answers generated via Alpha-Beta-Prune

3XO Project
 |-> nnTrainingData.txt
     |-> Python Project
	 |-> wih.csv, who.csv
 	     |-> 3XO Project loads wih.csv, who.csv ans initializes NN
